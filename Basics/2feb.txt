//DOM (Document Object Model)

DOM is how JavaScript talks to HTML.
HTML file → static text
Browser → converts HTML into a tree structure
This tree = DOM

JavaScript does NOT directly modify HTML files.
It modifies the DOM in memory, and the browser updates the screen.
